Newsletter

Newsletter form supports such services: Mailchimp, GetResponse, AWeber, Campaign Monitor, iContact. By defaulf e-mails from the form are written in the file php/emails.txt. To bind the form from these services, you need to make the line as a comment in the file php/notify-me.php.

define('FL_MAIL', 'emails.txt');

and uncomment the needed form (also adding the data from your Subscribe Form account).

Retina

In Xblog to enable the Retina for images you should have two images:

  • first one has the standard size and a standard name (image.jpg)
  • second one must be twice bigger then the origin and have additional annex @2x (image@2x.jpg)

Both images must be in the same folder!

So, how we can enable it in our html template? You should insert a class retina into the 'img' tag.

<img class="retina" src="image.jpg" width="158" height="158" alt="">
